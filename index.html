A tag de comentário é usada para inserir um comentário no código fonte HTML. Um comentário será ignorado pelo navegador.

Você pode usar comentários para explicar o seu código, o que pode ajuda-lo quando você editar o código fonte numa data posterior.

Tag Descrição


<html> Define um documento HTML
<body> Define o corpo do documento
<h1> a <h6> Define título 1 a título 6
<p> Define um parágrafo
<br/> Insere uma única quebra de linha
<hr/> Define um filete horizontal
<!--> Define um comentário
